# Heritage Unified — সেটআপ নির্দেশিকা

## এই অ্যাপটা কী
একটা নতুন, সম্পূর্ণ আলাদা গেটওয়ে অ্যাপ। বর্তমান হোস্টেল অ্যাপ বা HAIS-এর একটা ফাইলও এটা স্পর্শ করে না।

## চালু করার আগে ৩টা জিনিস ঠিক করতে হবে (কোডে TODO চিহ্নিত করা আছে)

### ১. হোস্টেল প্রজেক্টের Firebase `apiKey`
`index.html`-এ `HOSTEL_FIREBASE.apiKey` — এখন একটা placeholder বসানো আছে। Firebase Console →
`heritage-hostel` প্রজেক্ট → Project Settings → General → Web API Key থেকে আসল key কপি করে বসান।

### ২. দুই অ্যাপের হোস্টিং URL
`CONFIG.HOSTEL_APP_URL` ও `CONFIG.HAIS_APP_URL` — যেখানে বর্তমান দুই অ্যাপ লাইভ আছে সেই আসল URL বসান
(যেমন GitHub Pages লিংক বা Firebase Hosting লিংক)।

### ৩. প্রথম অ্যাডমিন রোল ম্যানুয়ালি তৈরি করা
রোল ম্যানেজমেন্ট UI ব্যবহার করতে প্রথমে অন্তত একজন অ্যাডমিন থাকা দরকার (chicken-and-egg সমস্যা এড়াতে)।
Firebase Console → `abu-sayed-6c6b9` প্রজেক্ট → Realtime Database → গিয়ে ম্যানুয়ালি এই এন্ট্রি যোগ করুন:

```
app_roles/
  আপনার_ইমেইল_আন্ডারস্কোর_দিয়ে/
    email: "আপনার@ইমেইল.com"
    canAccessHostel: true
    canAccessAcademic: true
    isAdmin: true
```

(ইমেইল কী তৈরির নিয়ম: ছোট হাতে করে, "." কে "_" দিয়ে ও "@" কে "__at__" দিয়ে বদলান —
যেমন `admin@school.com` → `admin__at__school_com`)

এরপর যেকোনো ব্রাউজারে `index.html` খুলে সেই ইমেইল/HAIS-পাসওয়ার্ড দিয়ে লগইন করলেই "রোল ম্যানেজমেন্ট"
ট্যাব দেখা যাবে — সেখান থেকেই বাকি সবাইকে যোগ করা যাবে, কোনো কোড না ছুঁয়ে।

## এই ভার্সনে যা আছে
- একক লগইন (HAIS-এর ইমেইল/পাসওয়ার্ড দিয়ে), রোল অনুযায়ী মডিউল দেখানো
- ইউনিফাইড ছাত্র প্রোফাইল ডিরেক্টরি — দুই ডেটাবেস লাইভ পড়ে মিলিয়ে দেখায়, কে কোন সিস্টেমে আছে/নেই তা নির্দেশ করে
- অ্যাডমিন রোল ম্যানেজমেন্ট UI
- **PWA (ফোনে ইনস্টলযোগ্য অ্যাপ)** — `manifest.json`, আইকন, ও সার্ভিস ওয়ার্কার যোগ করা আছে।
  Android/Chrome-এ প্রথমবার খুললেই "ইনস্টল করুন" ব্যানার দেখাবে (ব্রাউজার নিজে থেকেও ঠিকানা-বারে
  ইনস্টল আইকন দেখাতে পারে)। iPhone/Safari-তে ব্রাউজার নিজে থেকে প্রম্পট দেয় না — তাই শেয়ার (⬆️)
  বাটন থেকে "Add to Home Screen" নির্বাচন করার নির্দেশনা দেখানো হয়। ইনস্টল করার পর এটা হোমস্ক্রিনে
  একটা আলাদা আইকন হিসেবে বসবে এবং পূর্ণ-স্ক্রিন অ্যাপের মতো খুলবে (ব্রাউজার বার ছাড়া)।

## এই ভার্সনে যা নেই (পরবর্তী ধাপ)
- হোস্টেল স্টাফদের (যাদের HAIS ইমেইল নেই) জন্য একীভূত লগইন — আপাতত তাদের একটা HAIS ইমেইল
  অ্যাকাউন্টও লাগবে এই শেল অ্যাপে ঢুকতে
- ছাত্রের বিস্তারিত হাজিরা/ফলাফল/আচরণ ডেটা সরাসরি এই ড্যাশবোর্ডে দেখানো — আপাতত সেটার জন্য
  মূল অ্যাপে (এক ক্লিকে) পাঠিয়ে দেওয়া হচ্ছে, যাতে জটিল হিসাব দুইবার ভুলভাবে পুনর্লিখন না হয়
